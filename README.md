# manhuagui-dlr

純Python製作，執行過程不需要任何外部JS環境

包含自動webp->jpg，可選

初始化完後就能完全放置，過程中只有主控台視窗，可以繼續使用電腦不受影響

執行完後可參考[這裡](https://github.com/HSSLC/kc-generator)做下一步處理

required:  
* lzstring
* requests
* Pillow
* BeautifulSoup4

僅供學術研究交流使用，勿作為商業用途
